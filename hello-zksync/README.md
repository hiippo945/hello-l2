### Install

```shell
npm i
```

### Compile

```shell
# 1. Create .secret and paste your private key

# 2. Complie contract. Make sure Docker is enabled before running the command below
npx hardhat compile
```

### Deploy

```
npx hardhat deploy-zksync
```
